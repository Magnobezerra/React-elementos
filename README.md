# Estudos React

Projeto desenvolvido para praticar os conceitos fundamentais do React utilizando Vite.

## Tecnologias

* React
* JavaScript
* Vite
* CSS

## Objetivos do Projeto

* Aprender a criar componentes
* Trabalhar com props e estado
* Entender o ciclo de renderização do React
* Praticar organização de código

## Como executar

```bash
npm install
npm run dev
```

O projeto será iniciado em ambiente local através do Vite.
